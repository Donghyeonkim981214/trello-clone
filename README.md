# Drello

Drello is a project that clones Trllo.
Drello는 Trllo를 클론한 프로젝트입니다.

## Tech

Drello used the following techniques:

Backend
- python_version = "3.6"
- djangorestframework
- django-rest-knox
- django-cors-headers

FrontEnd
- reactJS
- react-redux
- bootstrap

Editor
- visual studio code

Deploy
- AWS EB, S3, RDS
- Web url: <del>http://drello-env.eba-jj6gw9st.ap-northeast-2.elasticbeanstalk.com/</del>(AWS요금부과 때문에 내려둔 상태)
